Weather App Suite — 7 языков, прогноз погоды по API
Коллекция из семи реализаций погодного приложения, использующих публичное API (OpenWeatherMap). Каждая версия включает:

Поиск погоды по названию города

Отображение: температура (°C/°F), влажность, скорость ветра, давление, иконка погоды

Прогноз на 5 дней (с интервалом 3 часа)

Автоматическое определение местоположения (по IP или геолокации в браузере)

Сохранение избранных городов (localStorage / JSON файл)

Переключение единиц измерения (метрические / имперские)

Цветовая индикация температуры (холодно/тепло/жарко)

График температуры (минимальный вариант — текстовый или простой canvas)

API Key
Для работы приложения требуется бесплатный API ключ от OpenWeatherMap. Получите ключ и установите его в переменную окружения OPENWEATHER_API_KEY или в конфигурационный файл.

Языки и технологии
Язык	Интерфейс	Библиотеки / фреймворки	Особенности
Python	CLI / Tkinter	requests, PIL	Автоопределение города по IP, график в консоли (ASCII)
JavaScript	Веб (HTML/CSS)	Fetch API, Chart.js	Геолокация браузера, анимация, адаптивный дизайн
Go	CLI / веб	net/http, html/template	Вывод цветной таблицы в консоли, кэширование запросов
Rust	CLI	reqwest, serde_json, colored	Цветной вывод, обработка ошибок, async
Java	Swing GUI	java.net.http, org.json	Иконки из URL, автоматическое обновление, трей
C#	Windows Forms	System.Net.Http, Newtonsoft.Json	График в PictureBox, сохранение избранных в XML
PHP	Веб	curl, file_get_contents	Поддержка сессий для избранных, кэширование ответов
